# Healthcare Data Analysis – IVF & Pregnancy Outcomes

## Project Overview

This project focuses on analyzing IVF and pregnancy outcome data to identify trends and factors affecting treatment success. The project demonstrates data cleaning, SQL analysis, exploratory data analysis (EDA), and interactive dashboard development using Python and Power BI.

## Tools Used

- Excel
- SQL
- Python
- Power BI

## Key Tasks

- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Querying
- Dashboard Development
- KPI Reporting
- Data Visualization

## Project Highlights

- Analyzed 600+ patient records with 70+ features.
- Cleaned and transformed healthcare datasets.
- Performed SQL analysis to extract treatment insights.
- Built an interactive Power BI dashboard with KPIs and visualizations.
- Presented findings through reports and dashboards to support data-driven decision-making.
